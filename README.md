# python-gui-calculator

This is a basic GUI Calculator built using Python and Tkinter.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Brackets ()
- Decimal (.)
- AC (All Clear)
- DEL (Single Delete)
- Error Handling

## Technologies Used
- Python
- Tkinter

## How to Run

1. Install Python
2. Download this project
3. Open terminal in project folder
4. Run:

python simple_calculator.py
