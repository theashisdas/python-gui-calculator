# python-gui-calculator

This is a basic GUI Calculator built using Python and Tkinter.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Brackets ()
- Decimal (.)
- AC (All Clear)
- DEL (Single Delete)
- Error Handling

## Technologies Used
- Python
- Tkinter

## Important Note
This is a desktop-based GUI calculator developed using Python Tkinter.
Since it is not a web application, it cannot be previewed directly on GitHub.

## To run this project:
1. Install Python
2. Download or clone this repository
3. Open terminal in the project folder
4. Run: python calculator.py
